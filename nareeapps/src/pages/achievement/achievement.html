How To Input Achievement

Berikut adalah langkah-langkah untuk memasukkan prestasi / achievement kedalam profile anda :


1. Pastikan anda masih memiliki foto sertifikat dan

foto anda bersama sertifikat saat event tersebut.

2. Kirimkan 2 foto tersebut ke email berikut:

naree.app@gmail.com

3. Gunakan subject "Input Achievement-(username)"

4. Tuliskan pada deskripsi email data berikut:

- Username ID    : (username naree anda)

- Nama Event      :

- Tanggal Event  :

- Tempat Event  : (dilengkapi nama kota)

- Peringkat         : 1/2/3(pilih salah satu)

- Tingkat Event  : lokal/nasion/internasional

(pilih salah satu)

Admin akan memverifikasi terlebih dahulu achievement yang anda input. Jika lolos, data dan bonus EXP akan otomatis ditambahkan pada akun anda.


note: Dilarang keras melakukan pemalsuandata achievement. Pelanggar akan dikenakan sanksi berupa banned account atau pengurangan EXP
